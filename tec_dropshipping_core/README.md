The logistics and financial framework for the Tecnopolis eCommerce Suite. Provides the base logic for multi-provider synchronization.

## Features
- **Multi-Location**: Map supplier warehouse columns to specific Odoo locations.
- **Financial Mapping**: Auto-conversion of ARS/USD costs and prices based on exchange rates.
- **Tax Mapping**: Standardized VAT conversion (10.5% / 21%) across different supplier formats.
- **Brand Mapping**: Robust matching of supplier brand names to local catalog brands.
